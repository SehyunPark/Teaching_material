1) 개발 환경 설치

(1)텍스트 에디터(프로그래밍 언어로 이루어진 코드를 작성)
(2)코드 실행기(코드를 실행)

2) 파이썬의 경우
(1)텍스트 에디터와 - 여기서는 Visual Studio Code를 다룸 
(2)코드 실행기(파이썬 인터프리터)를 합친 통합개발환경(IDE) 

3) 설치
(1) 텍스트 에디터인 Visual Studio Code 설치 - 구글 검색해서 Downloads 바로 클릭

(2) 파이썬 인터프리터 다운로드 - python 공식 사이트 - Downloads - Add Python 3.7 to PATH 체크 꼭

(3) 다섯번째 아이콘 확장 클릭 - Korean 검색 - Korean Language Pack 선택 - Install - 종료했다가 다시 실행

(4) 터미널 - 새 터미널 - 엔터 몇 번 누르고 python이라 입력 - 잘 설치되어있는 지 확인

4) 코드 작성, 실행하는 방법 (VS Code)
- 왼쪽 맨 위 아이콘 클릭 - 탐색기 - 폴더 열기 클릭 - 한 폴더 만들어서 클릭 
- 바로 아래에서 새 파일 클릭해서 파일을 생성하게 할 수 있다. (파이썬이면 .py)

5) 스네이크 케이스, 카멜 케이스 - 클래스, 변수(variable), 함수(function), 키워드 구분 + 주석
- 카멜은 각 단어를 대문자로 만들고 공백제거 (클래스)
- 스네이크는 밑줄 언더바 넣기 (함수 - 뒤에 괄호가 있으면, 변수 - 뒤에 괄호 없음)
- 뭔가 색이 다른 건 키워드
- 주석 설명 (# 그리고 ''' ''')
- 깃헙 아무 코드나 들어가서 한번 클래스, 함수, 변수, 키워드 구분 살펴보기

6) print(): 출력하고 싶은 내용을 집어넣음 - 여러개 넣을려면 사이에 콤마 삽입
- print(): 아무것도 입력 안한건 엔터 효과
- "안에 "를 출력하고 싶다면 "앞에 \를 표시하고 "붙이기
- 안에 \n을 넣으면 자동 다음줄로 넘어감

7) 나타내는 자료는 크게 문자열string, 숫자number, 불bool이 있음 (오늘은 문자열까지 - 다음에 숫자부터)
- 숫자는 int, float
- type()을 이용하면 자료형을 알 수 있음
- 문자열 연산자) + (문자열끼리만 가능) 안에 * 넣으면 배수만큼 문자열 여러개 출력 가능
(여기선 * 뒤에 숫자 사용)

8) 문자열 심화
- 문자 선택 연산자는 인덱싱, 문자열 범위 선택 연산자는 슬라이싱
- 인덱싱 [0]부터 시작 (거꾸로는 -1부터 시작)
- print('안녕하세요'[0]) print('안녕하세요'[-1])

- 슬라이싱은 :로 사용
- print('안녕하세요'[0:1]) 뒤쪽 숫자는 포함하지 않음 - 즉 0번째만 선택됨
- [2:] 와 [:2] 해보기, 거꾸로도 가능 [-2:-1]하나 어려워서 넘어감

- 문자열 길이도 구할 수 있음 len() 사용 - len()안에 문자열 넣으면 길이 출력
